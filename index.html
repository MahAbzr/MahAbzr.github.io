About

I’m a PhD student in Software Engineering at the University of Calgary, advised by Dr. Benjamin Tan and Dr. Lorenzo De Carli. My research looks at AI-assisted software development from a security angle: what happens when large language models generate code that looks right but isn’t.

Before Calgary, I worked on steganalysis and malware detection at Shiraz University, which is where I developed a taste for problems that live at the edge between machine learning and adversarial behavior.

Research

Secure code generation

Category-aware retrieval and prompting strategies for LLMs producing security-sensitive code.

Behavior of coding agents

Empirical analysis of how agents edit, rewrite, and repeat themselves across software engineering tasks.

Information security & steganalysis

Ensemble methods and entropy-based features for detecting hidden information and malicious binaries.
